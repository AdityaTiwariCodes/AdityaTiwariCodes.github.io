# 🚀 AI Assistant Setup

## 1. Install Dependencies

```bash
npm install express cors dotenv openai
```

---

## 2. Create `.env`

```env
OPENAI_API_KEY=your_openai_api_key
```

---

## 3. Run AI Server

```bash
node server.js
```
